嘿，我是 BINGHUAice — 喜欢折腾路由器，让它们干点原厂设计之外的事。

我在做什么

主要折腾这些：

  • OpenWrt / ImmortalWrt / iStoreOS — 各种路由器的第三方固件编译
  • Padavan — MTK 平台路由器的改版固件，正在搞 360 T6M 的适配
  • U-Boot (MT7621) — MTK SoC 平台的自定义引导加载程序
  • 路由器硬件改造 — 比如友华 WR1200JS 的 32MB 闪存改造

项目

  • padavan-4.4 — 添加了 360 T6M 支持的 Padavan 固件
  • uboot-mt7621 — MT7621 平台自定义 U-Boot
  • istoreos — WR1200JS 自用 iStoreOS
  • immortalwrt — 32M WR1200JS 自用 ImmortalWrt
  • Actions-OpenWrt — GitHub Actions 自动编译 OpenWrt
